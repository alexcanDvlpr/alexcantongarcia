Git es uno de los controles de versiones más usados, conocidos y demandados a día de hoy.

<YoutubeVideo youtubeId="kEPF-MWGq1w" youtubeTitle="Tutorial de Git" />

Los comandos básicos para empezar con git pueden ser los siguientes:

```bash
// Clonar Proyecto
git clone <url_proyecto> 

// Añadir cambios al index
git add .
```

Esto es código en **JavaScript**
```js
// Bucles y arrays
const fruits = ['apple', 'banana', 'watermelon']

console.log(fruits);

fruits = fruits.map(fruit => `${fruit}!`)

console.log(fruits) // apple!, banana!, watermelon!
```

Y esto es uun fragmento de **JSON**

```json
{
    "name": "Alex",
    "surname": "Cantón García",
    "birth": "25/11/1997"
}
```
